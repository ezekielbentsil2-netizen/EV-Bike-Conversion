# EV Bike Conversion

A hands-on project converting an internal combustion (IC) 
motorbike to an electric vehicle (EV), carried out under 
the guidance of a professional engineer.

## Project Overview
The goal was to strip the IC drivetrain from an existing 
motorbike chassis and replace it with an electric motor 
and custom battery system. The project was self-funded and 
self-sourced by the team. The bike had prior accident damage 
so the body was repaired using fibreglass before conversion.

## What We Did
- Sourced and acquired a damaged motorbike chassis
- Repaired the body using fibreglass
- Removed the internal combustion engine and drivetrain
- Built a custom battery system using lithium batteries
- Installed electric motor and integrated the battery system
- Handled all electrical wiring and connections
- Tested and validated the full conversion

## Key Concepts Applied
- Electric motor integration
- Custom lithium battery pack design and assembly
- Battery systems and power management
- Electrical wiring and circuit connections
- Fibreglass body repair and fabrication
- Mechanical and electrical system compatibility

## Tools & Skills Used
- Basic electrical tools
- Circuit wiring
- Fibreglass repair
- Component sourcing and research
